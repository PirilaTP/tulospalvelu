# Kirjoitin- ja tiedostotulosteet

Alasivulla *Kirjoitintulosteet* voidaan pyytää ratatiedoista neljän
eri tyypin tulosteita, joista piikkikuviot on graafinen tuloste ja muut
taulukkotulosteita radoista ja leimasinkoodeista.

Jos tietokoneeseen on asennettu tuki pdf-tiedostojen laatimiselle tulosteena,
on tämä erinomainen tapa tallentaa käytettäväksi graafiset piikkikuviot.
Pdf-tidostojen ohella voi tarjolla olla muitakin yhtä soveltuvia
tiedostomuotoja, kuten *Microsoft XPS Document* tai *One Note*.
Jos nämä mahdollisuudet ovat käytettävissä näkyvät ne kirjoitinvaihtoehtojen
joukossa.
