# Arvonta ja lähtöluettelot

Arvontaan liittyy kolme työvaihetta

- Arvonnan valmistelu

  - Lähtöaikojen antaminen

    - Kilpailijoiden numerointi

Kaikkiin työvaiheisiin pääsee valinnan *Valmistelu /
Arvonta ja numerointi* kautta, mutta ensin mainittu tehdään sarjatietojen
taulukossa, jonne voi siirtyä myös suoraan päävalikosta.

Kun kaikki nämä vaiheet on toteutettu, ovat
kilpailutiedot valmiina lähtöluetteloiden tulostusta ja itse kilpailua
varten.
