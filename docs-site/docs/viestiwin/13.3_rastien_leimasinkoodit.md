# 13.3 Rastien leimasinkoodit

### 13.3 Rastien leimasinkoodit

Tällä alalehdellä muutetaan rasteilla olevien
leimasimien koodeja. Kunkin rastin leimasimet ovat samalla rivillä. Yhdelle
rastille voidaan määritellä enimmillään kuusi koodia.

Tiedot tallennetaan tallentamalla ne ensin muistiin ja
sitten tiedostoon. Määritysten ristiriidattomuus voidaan tarkastaa, kun tiedot
on tallennettu muistiin.

Leimasinkoodit voivat sisältyä
samaan XML-tiedostoon kuin ratatiedot. Ohjelman tulospalvelutilassa on niiden löydyttävä samasta
tiedostosta ellei myös radoille käytetä vanhan tyyppistä tekstitiedostoa.

Kaikki leimasinkoodit voidaan poistaa painikkeella
*Tyhjennä taulukko*, minkä jälkeen voidaan uudet koodit lukea
yksinkertaisesta tekstitiedostosta, jonka kullakin rivillä on ensin rastikoodi
ja sitten yksi leimasinkoodi. Sama rastikoodi voi toistua monella
rivillä.
