# Luku 6. Emit-leimantarkastus, kuntosuunnistus

## 6 Emit-leimantarkastus, kuntosuunnistus

Ohjelma HkKisaWin sisältää
mahdollisuuden liittää siihen Emit-lukijaleimasimen tai muita Emit-laitteita.
Tässä vaiheessa on toimintoa kehitetty erityisesti kuntosuunnistusten
tarpeisiin. Ainakin suurissa kansallisissa suunnistuskisoissa suosittelen
käyttämään toistaiseksi varsinaiseen leimantarkastukseen ohjelmalla HkMaali.
Seuraavassa käsitellään kaikkia kuntosuunnistuksen toimintamoodeja kerralla.
Ohjeiden liiteluvuissa käydään yksi kerrallaan läpi, mitä kussakin
toimintamallissa on tehtävä.
