# 3.1 Ilmoittautumisten syöttö

### 3.1 Ilmoittautumisten syöttö

Valinnalla *Osanottajat / Yksi joukkue*
avautuvalla joukkuetietojen kaavakkeella valitaan *Tiedosto / Toistuvien
lisäysten tila*.

Tämän jälkeen syötetään tiedot ja tallennetaan ne.
Ohjelma tyhjentää kaavakeen uutta lisäystä varten.

Lopuksi poistetaan ruksi valinnasta *Toistuvien
lisäysten tila.*

Lisää tietoa kaavakkeen käytöstä on [luvussa 7.1](7.1_joukkue-_ja_henkilotiedot.md).
