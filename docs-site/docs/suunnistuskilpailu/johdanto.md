# Esimerkkikilpailun kuvaus

## Suunnistuskilpailun tulospalvelun järjestäminen

#### 2.5.2013

#### Pienehkö esimerkkikilpailu

Tässä ohjeessa käydään läpi tyypillisen kansallisen suunnistuskilpailun
tulospalveluun liittyvät työvaiheet.

Ennen näiden työvaiheiden aloittamista on asennettava ohjelmisto ja sen
oletetaan tapahtuneen asentamista koskevan ohjeen mukaisesti.

Esimerkkinä tarkasteltavaan kilpailuun odotetaan muutamaa sataa osanottajaa. Erillistä
ajanottoa ei ole, vaan ajat saadaan leimantarkastuksesta. Leimantarkastuksen
tietokone hoitaa lisäksi taustalla joitain muita tehtäviä. Itse
tulospalvelussa käytetään myös toista tietokonetta erilaisten muutosten
kirjaamiseen, itkumuuritoimintaan sekä kaikkiin tulospalvelun sekalaisiin
tehtäviin. Kolmas tietokone on kuuluttajalla kilpailun seuraamista varten.
