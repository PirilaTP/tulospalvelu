# 4.5 Sarjojen jakaminen

### 4.5 Sarjojen jakaminen

Alalehdellä *Sarjojen jakaminen* voidaan toteuttaa sarjan jakaminen useisiin sarjoihin. Tätä
tarvitaan tyypillisesti karsintasarjojen muodostamisessa kun osanottajat on
kirjattu yhteen sarjaan. Kilpailussa, jossa sarjat vaihtelevat, voi jakaminen
koskea vain yhtä vaihetta, esimerkiksi vain karsintavaihetta. Jakaminen voi
perustua arvontaan tai rankiin. On myös mahdollista jakaa sarja seurojen
perusteella, joko jakaen kukin seura eri alasarjoihin tai pitäen koko
seura mahdollisuuksien mukaan samassa alasarjassa.

Jakaminen toteutetaan valitsemalla ensin jaettava sarja
ja sitten muodostettavat sarjat. ohjelma kertoo jakamiseen liittyvää
lukumäärätietoa ja antaa mahdollisuuden muokata
sitä ennen jakamisen lopullista vahvistamista.
