# HkKisaWin-referenssi

HkKisaWin-ohjelman täydellinen käyttöohje, noin 120 sivua jaettuna lukuihin.
Tämä on **referenssidokumentaatio** — käytä sivunavigaatiota hakiessasi
yksityiskohtaista tietoa tietystä toiminnosta.

Jos olet uusi käyttäjä, aloita mieluummin
[Asentaminen ja tutustuminen](../asentaminen/yleista.md) -osiosta tai
lajikohtaisesta tutoriaalista.

Pääsivu: [HkKisa Help](hkkisahelp.md).
