### 5.7  Erän lähtöajan käsittely

Kun sarjan lähdöt tapahtuu erittäin ennalta tarkoin määrämättömän
aikataulun mukaisesti voidaan haluttu määrä lähtöaikoja määrittää lähtökellon
signaalista. Tämän jälkeen on erään osallistuvien joukkueiden numerot syötettävä
käsin kunkin ajan kohdalle. Menettely on parasta hoitaa niin, että lähdöt
käsitellään eri ajanottojonona kuin loppuajat ja väliajat. Kaikki jonot voidaan
käsitellä helpoimmin ohjelman ViestiWin erillisissä ajanottokaavakkaiessa tai
ohjelman ViestiMaali eri instansseissa. Ajanotossa voidaan käyttää esimerkiksi
seuraavia parametreja

```
// Timy portissa 4
timy=4
// Timyn START antaa lähtöajan STOP maaliajan ja kanava 2 väliajan 1
pisteet=MMMLM1
// jono1: maaliajat, jono 2: lähtöajat, jono 3: väliajat
jonot=111213
// jos aikoja käsitellään useammassa instanssissa, on ne lähetettävä antaen 
// parametri LÄHAIKAx käytetyille tiedonsiirtoyhteyksille
// ohjelmaa ViestiMaali käytettäessä kannattaa antaa maalin ja lähdön numeroiden 
// syötössä käytetyille koneille antaa parametrit
// piste=M
// piste=L
// erän koko 6 joukkuetta
lähtöryhmä=6
```

Kun lähtöaika kirjataan joukkueelle, merkitsee ohjelma
sen yksilölliseksi lähtöajaksi. Lähtökelloa voidaan käyttää myös myöhempien
osuuksien yksilöllisten lähtöjen yhteydessä. Lähtöaika ilmaistaan näytöillä
pisteenä 'L'.
