# 8.1 Yleistä

### 8.1 Yleistä

Ohjelma ViestiWin pohjautuu osittain aiempaan ohjelmaan
WinKuulutus ja sisältää kaikki sen toiminnot. Käytettävissä on useita eri
seurantanäyttöjä, joista monista voidaan käyttää samanaikaisesti useita
ikkunoita (yleensä 6). Täten voidaan seurata esimerkiksi useita sarjoja tai
kilpailijoita omissa ikkunoissaan.
