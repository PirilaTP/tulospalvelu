# Tulospalvelun konfigurointi

Tulospalvelun hoitaminen edellyttää
konfiguraatiotiedoston laatimisen, jos käytössä on tiedonsiirto tietokoneiden
välillä tai maalikello. Toimittaessa yhdellä tietokoneella ottaen ajat
tietokoneen kellosta eo konfiguraatiotiedosto ole välttämätön.

Konfiguraatiotiedoston laatimiseen on käytettävissä
konfiguraatioeditori, jonka saa esille ohjelman valinnassa *Tiedostot /
Avausmääritykset* ja sieltä edelleen *Avaa konfiguraatioeditori*.
