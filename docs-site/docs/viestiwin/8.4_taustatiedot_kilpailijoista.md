### 8.4 Taustatiedot kilpailijoista

Yksittäisen kilpailijan tietojen kaavakkeella voidaan
pyytää taustatietojen näyttöä. Tämä edellyttää, että tiedot ovat nettiselaimelle
sopivassa muodossa joko tiedostossa käytettävällä koneella tai lähiverkossa tai
internetpalvelimella osoitteessa, johon saadaan yhteys.

Käyttöä varten on laadittava tiedosto `tietolista.txt`, joka sisältää linkit kunkin henkilön
tietoihin. Tiedoston on oltava muotoa:

```
tiedosto    tietoa.html
2345        #mattila_matti15657http://www.ssl.fi/SSL/sslwww.nsf/sp?Open&cid=UrheilijaProfiili-2&id=05C10B813E919FA8C22570B4003619E4
56789        file:///c:/hakemisto/ali/tiedosto.html#kohta
nor00001234  #norjalainen
```

Ensimmäisellä rivillä kerrotaan tiedosto tai url -linkki sivulle, jolta
löytyvät myöhemmät lyhyet linkit. Rivien alussa olevat numerot ovat
lisenssinumeroita tai kansainvälisiä (WRE tai FIS) koodeja, joiden alussa on
maatunnus ja joiden on oltava kahdeksannumeroisia. Muilla riveillä on joko näitä
lyhyitä linkkejä ensimmäisen rivin sivulle tai täysiä linkkejä nettiin tai
muihin tiedostoihin. Paikallinen html-tiedosto voi olla hyvin pelkistetty ja
muotoa:

```
<html>
<body>
<h2>Tietoa kilpailijoista</h2>
<hr>
<h3><a name="mattila_matti">Matti Mattila</h3>
<p>Tietoa Matti Mattilasta
<hr>
<h3><a name="norjalainen">Norjalainen</h3>
<p>Tietoa Norjalaisesta
</body>
</html>
```
