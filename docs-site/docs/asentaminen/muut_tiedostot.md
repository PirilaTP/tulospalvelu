# Muut tiedostot

Ohjelmatoimitukseen sisältyy myös monia muita tiedostoja

- suunnistuksen seuraluettelo *seurat.csv* , josta saadaan lyhenteitä vastaavat täydet seuranimet

  - useita tekstitiedostoja, joida voidaan käyttää
    pohjana eräiden toimintojen, kuten esimerkiksi ulkoisen tietokannan käyttöönotossa

    - esimerkkimateriaalia, jota voi käyttää ohjelmiin
      tutustuttaessa ja josta löytyy myös hyödyllinen malli suunnistuksen sarjatiedoista
