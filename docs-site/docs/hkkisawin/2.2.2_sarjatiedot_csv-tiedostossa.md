# 2.2.2 Sarjatietojen käsittely csv-tiedostossa

#### 2.2.2 Sarjatietojen käsittely csv-tiedostossa

Sarjatiedot voi kirjoittaa csv-tiedostoon, jota on helppo
tarkastella ja myös muokata Excelissä. Muokatun tiedoston voi lukea takaisin
ohjelmaan. Tällöin on kuitenkin säilytettävä huolellisesti ohjelman kirjoittaman
tiedoston muoto. Ohjelma hyväksyy tiedostoa lukiessaan erotinmerkiksi sekä
puolipisteen että tabulaattorimerkin. Aivan kaikkia tietoja ei välttämättä
kirjoiteta ja lueta, joten tämä toiminto on tarkoitettu lähinnä tietojen
muokkaamiseen muilla apuvälineillä, kuten Excelillä
