# 5.9  Käyttö lähtökellona

### 5.9 Käyttö lähtökellona

Ohjelmaa voi käyttää lähtökellona seuraavasti

- Avataan tilanteen seurantakaavake

  - Valitaan näytettävä sarja ja näytettäväksi pisteeksi
    *Lähtö*

    - Valitaan näytettävät sarakkeet, merkkikoko, jonka
      osalta sopivin saattaa löytyä valinnan *Muu merkkikoko* kautta, sekä sarakkeiden leveydet.

      - Valikosta valitaan *Seuraa lähtöjä* sekä
        haluttaessa äänimerkkien lukumäärä (esim. 1: vain lähtöhetkellä, 4: kolme
        lyhyttä merkkiä sekunnin välein ja pitempi lähtöhetkellä)

Ohjelma näyttää aina vain valitun sarjan tai
sarjayhdistelmän lähtijät ja antaa äänimerkit vain tällaisten lähtijöiden
kohdalla. Kaikki lähtöpaikan sarjat saadaan mukaan määrittelemällä
sarjayhdistelmä, joka sisältää kaikki nämä sarjat (katso [luku 2.3](2.3_sarjayhdistelmat.md)
).
