# Luku 2. Kilpailun luominen ja määrittely

## 2 Kilpailun luominen ja määrittely

Kilpailun luomiseen sisältyy

- kilpailutiedoille käytettävän kansion valinta ja mahdollinen luominen

  - kilpailun perustietojen määrittely

    - sarjatietojen määrittely määritellen kaikkien
      sarjojen nimet, osuuksien lukumäärät
      ja mahdollisesti muita tietoja

      - ohjelman käynnistäminen esivalmistelumoodissa, jolloin ohjelma luo uuden kilpailijatietotiedoston.
