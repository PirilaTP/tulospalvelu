# 9.8 Kilpailijakohaiset tulosteet

### 9.8 Kilpailijakohaiset tulosteet

Suunnistuskilpailuissa voidaan tulostaa
kilpailijakohtaisesti rastiväliaikatulosteita ([luku 6.8](6.8_rastivaliaikatulosteet.md) ) sekä
leimantarkastustulosteita ([luku
6.1](6.1_leimantarkastuskaavake.md)
).

Tulostuskaavakkeen valinta *Tiedosto / Kilpailijan
sivutuloste* avaa kaavakkeen, jolla voidaan määritellä ja valita
tulostettavaksi muutamia kilpailijakohtaisia tietoja kirjoittimen sivulle. Tämä
toiminto on tarkoitettu käytettäväksi esipainettujen lomakkeiden kanssa
esimerkiksi
osanottotodistusten tulostamiseen.

Käytgettävissä olevien tietojen osalta voidaan valita
niiden sijainti sivulla. Vaakasuunnassa sijainti ilmoitetaan sivun leveyden
tuhannesosina ja pystysuunnassa pisteinä eli yksikössä 1/72 tuumaa. A4-arkille
mahtuu noin 1100 pistettä. Sijainnin lisäksi voidaan määritellä tekstin maksimi
merkkimäärä, kirjaisinkoko muotoilumääritysten
mukaisista vaihtoehdoista:

1: normaali sisältökirjasin  
2: alaotsikoiden
kirjasin  
3:
pääotsikon kirjasin

Edelleen voidaan valita, tasataanko teksti vasemmalle
vai oikealle.

Tulostus
voidaan tehdä

- yksittäiselle kilpailijalla valitsemalla
  numeron mukaan
- yhden sarjan
  kaikille kilpailijoille
- kaikille
  läsnäoleviksi merkityille
- automaattisesti aina, kun aiemmin vakantiksi tai
  ei-lähteneeksi merkitty osanottaja
  vaihdetaan läsnäolevaksi.

Kun määritykset on tehty, voidaan ne tallentaa
tiedostoon, jonka oletusnimi on PgTulMaar.xml. Jos
tämän niminen tiedosto löytyy kilpailutietojen hakemistosta avattaessa kaavake
ensimmäistä kertaa, lukee ohjelma
sen automaattisesti.
