# Luku 4. Arvonta ja kilpailunumeroiden määrääminen

## 4 Arvonta ja kilpailunumeroiden määrääminen

Arvontatoiminto on käytettävissä esivalmistelutilassa
sekä useampivaiheisessa kilpailussa käynnissä olevaa vaihetta myöhempien
vaiheiden lähtöaikojen ja numeroiden määrittämiseen.

Päätoimintoja on kolme:

- osanottajien jako alisarjoihin, kuten karsintasarjoihin

  - lähtöaikojen arvonta

    - kilpailunumeroiden antaminen

Kaavakkeen otsikot kertonevat eri valintojen merkityksen. Lisää
ohjeita on lajikohtaisissa tulospalvelun ohjeissa.
