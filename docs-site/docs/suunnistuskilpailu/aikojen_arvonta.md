# Lähtöaikojen arvonta

Arvontaan siirrytään valinnan *Valmistelu / Arvonta ja numerointi* kautta. Lähtöajat
arvotaan kaavakkeen *Arvonta ja numerointi*
alasivulla *Arvonta.*

Tavanomaisen suunnistuskilpailun tapauksessa ei edellä
kuvatun esivalmistelun jälkeen tarvitse tehdä muuta kuin

- valita *Arvonta
  suunnistuksen säännöin* ,

  - klikata painiketta *Lisää
    kaikki* ja lopuksi

    - *Toteuta arvonta.*

Kaavakkeen valinnoista *Arvonta
suunnistuksen säännöin* merkitsee sitä, että ohjelma estää saman seuran
jäsenten lähtemisen peräkkäisillä lähtöajoilla, kun se suinkin on mahdollista.
Kun käytettäväksi seuranimeksi valitaan *Lyhenne*,
käyttää ohjelma seuralyhennettä tässä tarkastuksessa. Esto ei välttämättä toimi,
jos osalla osanottajista on kirjattuna maakoodi ja osalla ei.
