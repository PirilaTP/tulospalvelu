# 1.3.5 Valinta: Tulospalvelu

Valinnasta *Tulospalvelu* siirrytään ajantoton
sekä leimantarkastuksen toimintoihin. Tästä valinnasta pääsee myös pisteiden
laskentaan tulosten perusteella kilpailun loppuvaiheessa sekä muuttamaan
sarjoittain lähtöaikoja, jos kilpailun aikataulu on muuttunut.

*![](../images/ValikkoTulospalvelu.png)*

- **Ajanotto, Ajanotto - 2.** Kaavake
  ajanoton seurantaan. Mahdollisuus seurata kahta eri
  ajanottotaulukkoa.
- **Emit-luenta.** Emit-leimantarkastus,
  joka voi sisältää myös tulosten laskennan.
- **Laske kilpailijoiden pisteet.**
  Kaavake pisteiden laskemiseksi aikojen tai sijoitusten perusteella, kun
  tulokset ovat riittävän valmiit.
- **Muuta sarjan lähtöaikoja.**
  Lähtöaikojen siirto, kun kilpailun aikataulu on muuttunut tai yhteislähtö
  tapahtunut ennakkotiedosta poikkeavalla hetkellä.

Edellä luetelluista valinnoista on Emit-luenta
käytettävissä vain, kun Emit-toiminnot on käynnistetty. Lajin ollessa ampumahiihto
tulee valittaviin vaihtoehtoihin lisäksi

- **Ammunnan sakot.** Tässä valinnassa voidaan syöttää ohilaukausten määrät sekä seurata Kurosen
  laitteistolta saatuja tieoja ohilaukauksista.
