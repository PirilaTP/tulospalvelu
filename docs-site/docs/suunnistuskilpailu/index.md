# Suunnistuskilpailun tulospalvelu

Alkuperäinen tutoriaali ensimmäisen suunnistuskilpailun tulospalvelun
järjestämiseen Pekka Pirilän ohjelmistolla. Sisältää kattavat ohjeet aloituksesta
tulosten julkaisuun.

Navigaatiosta vasemmalta löytyvät täydelliset ohjeet lukukohtaisesti. Aloita
[Esimerkkikilpailun kuvaus](johdanto.md) -sivulta.
