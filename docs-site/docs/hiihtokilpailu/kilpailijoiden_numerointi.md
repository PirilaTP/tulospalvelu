# Kilpailijoiden numerointi

Kun lähtöajat on arvottu, siirrytään alisivulle *Numeroiden antaminen.*
Jos tämä tehdään heti lähtöaikojen arvonnan perään, ovat sarjat valmiiksi
valittuina eikä tarvitse tehdä mitään muuta kuin todeta, että valinnat ovat
halutut, kuten niiden pitäisi olla, ja klikata painiketta *Toteuta
numerointi*.
