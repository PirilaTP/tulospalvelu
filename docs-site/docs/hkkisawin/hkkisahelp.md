# HkKisa Help

## Henkilökohtaisen kilpailun tulospalveluohjelmat HkKisaWin ja HkMaali

### Versio 5.20

### Pekka Pirilä, 19.4.2015

Graafista Windows-käyttöliittymää soveltava ohjelma
HkKisaWin soveltuu lähes kaikkien kilpailun tulospalveluun liittyvien
toimintojen hoitamiseen. HkMaali on pelkistetympi komentoikkunaohjelma, jonka
luotettavuutta on maksimoitu välttämällä monimutkaisten
käyttöliittymäkirjastojen ja kehitysvälineiden käyttöä. Siksi HkMaali sopii
paremmin merkittävien kilpailutapahtumien kriittisimpien reaaliaikaisten
tehtävien hoitamiseen. Ohjelmat toimivat yhdessä niin, että kilpailun aikana
kukin tehtävä voidaan hoitaa tarkoitukseen sopivammalla ohjelmalla.

Näissä ohjeissa on pääpaino ohjelmalla HkKisaWin.
Ohjelman HkMaali käyttöä kuvataan mm. erillisessä pdf-tiedostona
toimitettavassa dokumentissa. Niitä erityispiirteitä, jotka sisältyvät vain ohjelmaan
HkMaali, käsitellään usein vain ohjelman HkMaali pdf-ohjeissa.

Lisää ohjelmien käyttöön liittyviä ohjeita
löytyy nettisivuiltani. Sieltä löytyy mm. ohjeita ohjelmien asentamiseen
sekä lajikohtaisia ohjeita tulospalvelun hoitamiselle ohjelmia käyttäen.
