# Ristiriidattomuus

Kaavakkeen painike *Tarkasta määritykset* saa ohjelman käymään läpi
tietokoneen muistiin tallennetut rata- ja rastileimasintiedot. Ohjelma tarkastaa
tällöin ensisijaisesti sen, että milllään radalla ei esiinny samaa leimasinta
peräkkäisillä rasteilla. Tällainen tilanne johtaisi siihen, että jälkimmäiseltä
rastilta ei saada lainkaan leimaa. Ohjelma huomauttaa myös jonkin verran
lievemmistä virheistä, jotka eivät tee leimantarkastusta täysin mahdottomaksi,
mutta saattavat sitä vaikeuttaa.

Tarkastuksen tulokset ovat luettavissa alasivulla *Tarkastuksen
tulos*.
