# 1.5  Kaavakkeiden asetusten tallentaminen

### 1.5 Kaavakkeiden asetusten tallentaminen

Kun tietokoneen rrudulle on avattu käytettävät
kaavakkeet sijoitettuna tavalla, jota halutaan jatkossakin käyttää, on tämä
tilanne mahdollista tallentaa pääkaavakkeen valinnassa *Tallenna
ikkunat*
. Tämä mahdollisuus koskee vain sellaiseia
kaavakkeita, joita käytetään usein toistuvasti:

- Sarjan tilanteen seuranta- Viimeiset tapahtumat- Kilpailun
      status- Emit-leimantarkastus- Sisäänluenta- Ajanotto

Samalla, kun ohjelma tallentaa kaavakkeiden sijoittelun,
tallentaa se myös useimmat kaavakkeilla tehtävissä olevat valinnat. Myös kahden ensinmainitun kaavakkeen taulukon
sarakkeiden leveydet tallentuvat edellyttäen, että nämä on ensin tallennettu
muistiin kyseisen kaavakkeen valikon kautta.

Muistiin tallennetut sijainnit ja
asetukset voidaan lukea joko valikon kautta tai automaattisesti, kun
tätä on pyydetty konfiguraatiotiedoston parametrilla

ikkunat=xxxx

missä xxxx on tiedot
sisältävän tiedoston nimi. Ellei nimeä anneta, käytetään nimeä ikkunat.xml.
