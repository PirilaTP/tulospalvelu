# Ratatiedot

Emit-leimausjärjetelmään perustuvassa kilpailussa tarvitaan tiedot radoista
ja niillä käytettävistä leimasimista. Nämä määritellään ohjelman valinnassa
*Valmistelu / Ratatiedot*. Muista tiedoista poiketen voidaan ratatietoja
käsitellä jo ennen koko kilpailun perustamista tai tietokoneella, jolla ei ole
käytettävissä muita tietoja kilpailusta, jos se katsotaan
hyödylliseksi.

Ratatietokaavakkella käsitellään
tietoja ensin niin, että ne tallennetaan tietokoneen muistiin, jossa niiden ristiriidattomuuskin
voidaan tarkastaa, ja sitten vasta erikseen pyydettäessä
tiedostoon.
